# 加载参考文档

将参考材料读取到当前会话上下文，供编写/修订时借鉴内容结构或素材。

## 用法

```
/load-refs /path/to/ref-file.md                       # 单文件
/load-refs /path/to/ref-file.md 用途：提供接口定义    # 注明用途（可选）
```

参数为文件路径（必填）。不支持目录模式；参考材料通常是外部来源，不在项目 `CLAUDE.md` 的关联关系中。

## 执行步骤

1.   读取指定文件
2.   提取用途说明（若参数中有说明则使用；否则标记为"未指定"）
3.   输出加载确认：
   ```
   [LOADED refs] {文件路径} | ~{字节数÷3.5取整} tokens  ← 参考材料，用途：{用途说明}
   ```

### 加载后

扫描本次会话全部 `[LOADED *]` 记录，列出已加载文件清单及估算合计。

容量检查（以 200K 为基准）：
-   合计 > 120K tokens → ⚠️ 已超过 60%，谨慎继续加载
-   合计 > 160K tokens → 🔴 已超过 80%，强烈建议停止加载

> 提示：参考材料通常较大，加载前建议先执行 `/cap-status` 确认剩余容量。
