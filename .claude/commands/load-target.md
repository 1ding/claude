# 加载目标文档

将指定文件读取到当前会话上下文，标记为目标文档。

## 用法

```
/load-target /path/to/file.md
```

参数为具体文件路径（必填，不支持目录）。

## 执行步骤

1.   读取 $ARGUMENTS 指定的文件
2.   输出加载确认：
   ```
   [LOADED target] {文件路径} | ~{字节数÷3.5取整} tokens
   ```
3.   扫描本次会话全部 `[LOADED *]` 记录，列出已加载文件清单及估算合计
4.   容量检查（以 200K 为基准）：
-   合计 > 120K tokens → ⚠️ 已超过 60%，谨慎继续加载
-   合计 > 160K tokens → 🔴 已超过 80%，强烈建议停止加载
